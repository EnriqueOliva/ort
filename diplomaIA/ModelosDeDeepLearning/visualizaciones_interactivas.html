ResNet (Residual Network)

Problema: Redes muy profundas sufren de vanishing gradient

Solucion: Skip connections que suman la entrada directamente

Bloque Residual Interactivo

ENTRADA (x)
|
Conv 1x1 (256 -> 64)
|
BatchNorm + ReLU
|
Conv 3x3 (64 canales)
|
BatchNorm + ReLU
|
Conv 1x1 (64 -> 256)
|
F(x)
SKIP
x
|
F(x) + x (SUMA)
|
SALIDA H(x)
Formula clave:
H(x) = F(x) + x

Por que funciona:
dH/dx = dF/dx + 1
El "+1" garantiza que el gradiente NUNCA desaparece

RNN (Recurrent Neural Network)

Procesa secuencias manteniendo un estado oculto

RNN Desplegada en el Tiempo

    H0 ----> [RNN] ----> H1 ----> [RNN] ----> H2 ----> [RNN] ----> H3
               ^                    ^                    ^
               |                    |                    |
              X1                   X2                   X3
               |                    |                    |
               v                    v                    v
              Y1                   Y2                   Y3

    Los pesos U, V, W se COMPARTEN en todos los pasos

Celda RNN

Xt (entrada)
Ht-1 (estado previo)
| |
tanh(U*Xt + V*Ht-1 + b)
|
Ht (nuevo estado)
|
W*Ht
|
Yt (salida)
Formulas:
Ht = tanh(U*Xt + V*Ht-1 + b)
Yt = W*Ht

LSTM (Long Short-Term Memory)

Resuelve el vanishing gradient con compuertas (gates)

Simulador de Compuertas LSTM

FORGET
0.70
INPUT
0.80
OUTPUT
0.60
Interpretacion:
- Forget=0.70: Mantiene 70% de la memoria anterior
- Input=0.80: La info nueva es muy relevante
- Output=0.60: Muestra 60% del estado actual
Formulas:
ft = sigmoid(Wf * [Ht-1, Xt])
it = sigmoid(Wi * [Ht-1, Xt])
Ct_tilde = tanh(Wc * [Ht-1, Xt])
Ct = ft * Ct-1 + it * Ct_tilde (Cell State)
ot = sigmoid(Wo * [Ht-1, Xt])
Ht = ot * tanh(Ct) (Hidden State)

GRU (Gated Recurrent Unit)

Simplificacion de LSTM: 2 compuertas en vez de 4

Simulador de Compuertas GRU

UPDATE (z)
0.30
RESET (r)
0.80
Interpretacion:
Ht = (1-z)*H_nuevo + z*Ht-1
Ht = 0.70*H_nuevo + 0.30*Ht-1

- z=0.30 -> 70% nuevo, 30% anterior
- Prioriza la informacion NUEVA
Aspecto LSTM GRU
Estados 2 (C y H) 1 (solo H)
Compuertas 4 2
Parametros Mas Menos
Entrenamiento Mas dificil Mas facil

Mecanismo de Attention (Bahdanau)

Permite al decoder "mirar" diferentes partes del input

Matriz de Attention Interactiva

Ejemplo: Frances -> Ingles. Click en una celda para ver el peso.

Haz click en una celda para ver el peso de atencion
Formulas:
e_ij = v^T * tanh(Ws*Si-1 + Wh*Hj)
alpha_ij = softmax(e_ij)
Ci = SUM(alpha_ij * Hj) (Contexto dinamico)

Self-Attention

Cada palabra "mira" a TODAS las otras palabras de la MISMA secuencia

Query, Key, Value

Input: [X1, X2, X3]
(Embeddings de palabras)
|
Q = X * Wq
Queries
K = X * Wk
Keys
V = X * Wv
Values
|
Scores = Q * K^T / sqrt(dk)
|
Softmax (por filas)
|
Output = Softmax * V
(Embeddings contextuales)
Formula:
Attention(Q,K,V) = softmax(Q*K^T / sqrt(dk)) * V
