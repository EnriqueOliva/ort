## **COMPOSE STRATEGIES**

### **OneOf** - Select exactly one transform
```python
A.OneOf([
    A.GaussianBlur(blur_limit=(3, 7), p=1.0),
    A.MotionBlur(blur_limit=7, p=1.0),
    A.MedianBlur(blur_limit=5, p=1.0),
], p=0.3)  # 30% chance to apply ONE of these blurs
```

### **SomeOf** - Select N random transforms
```python
A.SomeOf([
    A.Sharpen(p=0.5),
    A.RandomGamma(p=0.5),
    A.CLAHE(p=0.5),
], n=(1, 2), p=0.8)  # 80% chance to apply 1 or 2 of these
```

### **Sequential** - Apply all or none
```python
A.Sequential([
    A.CLAHE(p=1.0),
    A.Sharpen(p=1.0),
], p=0.5)  # 50% chance to apply BOTH
```

### **Ordering Best Practices:**
1. Resize first (smaller images = faster)
2. Geometric transforms early
3. Heavy transforms (distortion) in middle
4. Color/lighting after geometry
5. Normalize always last
