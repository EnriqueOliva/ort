Encoder-Decoder para Traducción Automática

💡 Concepto Clave
Encoder: Aprende una representación eficiente, comprime la información y la codifica. Encuentra una representación latente que se pasa al decoder.

Decoder: Aprende a decodificar esa información para generar la salida en el idioma objetivo.
⚠️ Importante: No es Recurrencia
Cada bloque tiene sus propios pesos independientes. No es como las RNN donde se reutilizan los pesos. Si N=2, son literalmente 2 capas diferentes de encoder y 2 capas diferentes de decoder, cada una con sus propios parámetros entrenables.
ENCODER
📝 Input: Frase en Español
↓
Input Embedding
Convierte cada palabra en un vector de dimensión d_model
Vocabulario × d_model
Ejemplo: 5000 × 128
+
Positional Encoding
Agrega información de posición porque el attention es invariante a permutaciones
↓
⟲ Se repite N veces (N = 2 en el ejercicio)
Bloque Encoder
1️⃣ Multi-Head Self Attention
Todas las palabras se miran entre sí. Captura relaciones semánticas y sintácticas.
➕ Add (Skip Connection) + Layer Normalization
2️⃣ Feed Forward Network
MLP de 2 capas: d_model → d_ff → d_model
Se aplica a cada palabra por separado
128 → 512 → 128
➕ Add (Skip Connection) + Layer Normalization
↓
📊 Output: Representación Latente
(N palabras × d_model)
DECODER
🎯 Input: Frase en Inglés
(durante entrenamiento)
↓
Output Embedding
Convierte cada palabra en un vector de dimensión d_model
Vocabulario × d_model
Ejemplo: 6000 × 128
+
Positional Encoding
Mismo propósito que en el encoder
↓
⟲ Se repite N veces (N = 2 en el ejercicio)
Bloque Decoder
1️⃣ Masked Multi-Head Self Attention
¡Con máscara! No puede ver el "futuro". Solo palabras anteriores.
➕ Add + Layer Normalization
2️⃣ Cross-Attention (Encoder-Decoder)
Conecta con el Encoder.
Query: del decoder | Keys y Values: del encoder
➕ Add + Layer Normalization
3️⃣ Feed Forward Network
Igual que en el encoder
128 → 512 → 128
➕ Add + Layer Normalization
↓
Capa Lineal
Proyecta a la dimensión del vocabulario
(d_model + 1) × Vocabulario
Ejemplo: 129 × 6000
↓
Softmax
Convierte a probabilidades sobre todo el vocabulario
↓
✨ Output: Frase Traducida
(probabilidad por palabra)
📏 Importante sobre Dimensiones
En el Encoder: Si entra una frase de longitud N, sale una matriz de N × d_model
En el Decoder: Si entra una frase de longitud M, sale una matriz de M × d_model
Las longitudes pueden ser diferentes: Una frase de 5 palabras en español puede traducirse a 8 palabras en inglés.
