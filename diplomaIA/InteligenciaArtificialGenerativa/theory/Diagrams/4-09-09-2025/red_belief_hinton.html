Red Belief Networks de Hinton

La idea de Geoffrey Hinton (Premio Turing 2018) fue organizar los perceptrones autorregresivos en una única red neuronal con una característica especial:

En lugar de tener perceptrones separados, todos se combinan en una red donde ciertos pesos están fijados en cero para mantener la dependencia autorregresiva.

Matriz de pesos W con estructura triangular inferior:
W = [0 0 0 0 ] [W₂₁ 0 0 0 ] [W₃₁ W₃₂ 0 0 ] [W₄₁ W₄₂ W₄₃ 0 ]
x̂ = σ(W · x + b)

Ventajas de esta arquitectura:

  • Entrenamiento en paralelo: Podemos pasar toda la imagen a la vez y calcular todas las predicciones simultáneamente
  • Menos parámetros: En vez de crecer exponencialmente (2ⁿ), crece linealmente (n)
  • Optimización eficiente: Usamos backpropagation sobre toda la red a la vez
  • Generación secuencial: Para generar/muestrear, seguimos usando el proceso secuencial
Entrenamiento: Paralelo (todas las salidas a la vez)
Loss = -Σᵢ [xᵢ·log(x̂ᵢ) + (1-xᵢ)·log(1-x̂ᵢ)]
Generación: Secuencial (un píxel a la vez)
x₁ ~ Bernoulli(x̂₁), luego x₂ ~ Bernoulli(x̂₂), etc.

Restricción autorregresiva:

Los pesos en rojo punteado están fijados en cero y NO se entrenan. Esto asegura que:

  • x₁ no depende de nada (solo de b₁)
  • x₂ solo depende de x₁
  • x₃ solo depende de x₁ y x₂
  • x₄ depende de x₁, x₂ y x₃
