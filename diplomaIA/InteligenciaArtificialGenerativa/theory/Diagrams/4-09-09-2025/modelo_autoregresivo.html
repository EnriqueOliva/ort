Modelo Autorregresivo

Un modelo autorregresivo genera datos de manera secuencial, donde cada valor depende de los valores anteriores.

Idea clave: En lugar de estimar directamente la distribución conjunta p(x₁, x₂, ..., xₙ), usamos la regla de la cadena:

p(x₁, x₂, ..., xₙ) = p(x₁) · p(x₂|x₁) · p(x₃|x₁,x₂) · ... · p(xₙ|x₁,...,xₙ₋₁)
= ∏ᵢ₌₁ⁿ p(xᵢ | x₁, ..., xᵢ₋₁)

Para imágenes binarias (blanco y negro):

  • Cada píxel xᵢ puede ser 0 (apagado) o 1 (prendido)
  • Modelamos cada píxel como una Bernoulli con parámetro αᵢ
  • El parámetro αᵢ depende de los píxeles anteriores
  • Usamos un perceptrón para estimar αᵢ
Estimación con perceptrón:
p(xᵢ = 1 | x₁,...,xᵢ₋₁) = σ(Wᵢ · [x₁,...,xᵢ₋₁] + bᵢ)
donde σ es la función sigmoide: σ(z) = 1/(1 + e⁻ᶻ)

Proceso de generación (muestreo):

  1. Empezamos con el primer píxel: muestreamos de p(x₁)
  2. Para el segundo: usamos x₁ para calcular p(x₂|x₁) y muestreamos
  3. Para el tercero: usamos x₁, x₂ para calcular p(x₃|x₁,x₂) y muestreamos
  4. Repetimos hasta completar todos los píxeles

Importante: El proceso es secuencial, no podemos generar todos los píxeles en paralelo porque cada uno depende de los anteriores.
