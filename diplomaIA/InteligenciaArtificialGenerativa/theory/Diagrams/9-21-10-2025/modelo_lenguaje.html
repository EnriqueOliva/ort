🤖 Modelo de Lenguaje (Language Model)

Clase 9 - 21 de Octubre 2025

¿Qué es un Modelo de Lenguaje?

  • Sistema que predice la próxima palabra en una secuencia de texto
  • Funciona como un clasificador que emite distribuciones de probabilidad
  • Es un modelo autorregresivo: usa las palabras anteriores para predecir la siguiente
  • Similar a cómo se predicen píxeles condicionados a los anteriores

📊 Arquitectura del Modelo de Lenguaje

Entrada

Secuencia de Texto

σ* (tokens)
→

Language Model

Procesa la secuencia completa

LM(·)
→

Salida

Distribución de Probabilidad

P(σ ∪ {EOS})

Definición Formal

LM : σ* → Δ(σ ∪ {EOS})

Donde σ es el conjunto de tokens, σ* son todas las combinaciones posibles, y Δ es el simplex de probabilidades

Ejemplo: "Hola, ¿cómo ___?"

El modelo genera probabilidades para cada token posible:

estás - 0.45 (45%)
andas - 0.20 (20%)
te va - 0.15 (15%)
va - 0.10 (10%)
? (EOS) - 0.05 (5%)
otros... - 0.05 (5%)

🔄 Proceso Autorregresivo

1
Entrada: "Hola, ¿cómo"
Salida: Distribución → Samplea "estás"
2
Entrada: "Hola, ¿cómo estás"
Salida: Distribución → Samplea "?"
3
Entrada: "Hola, ¿cómo estás?"
Salida: Distribución → Samplea "EOS" (fin)

Resultado final: Hola,¿cómoestás?

Características Importantes

  • Determinístico en la salida: Para la misma entrada, genera la misma distribución de probabilidad
  • Estocástico en la generación: El sampling introduce variabilidad
  • Largo máximo del contexto: No puede procesar secuencias infinitas (limitado por la arquitectura)
  • Token EOS: Indica el fin de secuencia, tiene su propia probabilidad
  • Entrenamiento: Aprende con descenso de gradiente y después con reinforcement learning (RLHF)

Proceso de Generación

1. LM(tira) → distribución
2. sample(distribución) → token
3. concatenar(tira, token) → nueva_tira
4. repetir hasta EOS o max_length
