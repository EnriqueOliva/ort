🎲 Estrategias de Sampling en LMs

Clase 9 - 21 de Octubre 2025

¿Qué es el Sampling?

  • Definición: Proceso de elegir el siguiente token basándose en la distribución de probabilidad que genera el modelo
  • No determinístico: Introduce aleatoriedad controlada en la generación
  • Control de calidad: Las estrategias de sampling afectan la coherencia y creatividad del texto
  • Parámetros clave: Top-K, Top-P, Temperatura, y max_length

🎯 Top-K Sampling

K = 3 (solo los 3 más probables)

estás 45%
andas 20%
te va 15%
va 10% ❌
otros 10% ❌

Cómo funciona:

1. Ordena tokens por probabilidad

2. Toma solo los K más probables

3. Elimina todos los demás

4. Re-normaliza probabilidades

5. Samplea de los K seleccionados

📊 Top-P (Nucleus) Sampling

P = 0.80 (acumulado hasta 80%)

estás 45% ✓ (total: 45%)
andas 20% ✓ (total: 65%)
te va 15% ✓ (total: 80%)
va 10% ❌ (90%)
otros 10% ❌

Cómo funciona:

1. Ordena tokens por probabilidad

2. Suma probabilidades acumuladas

3. Incluye tokens hasta llegar a P%

4. Elimina el resto

5. Re-normaliza y samplea

⚖️ Top-K vs Top-P: Comparación

Top-K = 3

Ventaja Número fijo de opciones

Desventaja Puede incluir tokens poco probables si K es grande

Uso Cuando quieres limitar opciones de forma estricta

Top-P = 0.80

Ventaja Adaptativo: más tokens si están distribuidos uniformemente

Desventaja Número variable de tokens

Uso Generalmente mejor calidad de texto

🌡️ Temperatura

¿Qué hace la Temperatura?

  • Modifica la softmax: Suaviza o agudiza la distribución de probabilidad
  • No es Top-K ni Top-P: Es independiente, pero se aplica ANTES
  • Valor típico: T = 1.0 (sin cambios)
  • T {'<'} 1: Más determinístico, menos creativo
  • T {'>'} 1: Más aleatorio, más creativo

Fórmula de Softmax con Temperatura

P(token_i) = exp(logits_i / T) / Σ exp(logits_j / T)

T = 0.5 (Baja)

Efecto: Distribución más "puntiaguda"

Siempre elige opciones seguras

T = 1.0 (Normal)

Efecto: Distribución original del modelo

Balance entre coherencia y variedad

T = 2.0 (Alta)

Efecto: Distribución más "plana"

Más creativo pero menos coherente

🛑 Max Length (Longitud Máxima)

  • Problema: El modelo podría no generar nunca el token EOS (End of Sequence)
  • Solución: Definir un máximo de tokens a generar
  • Condición de parada 1: El modelo genera EOS
  • Condición de parada 2: Se alcanza max_length tokens
  • Valor típico: 50-2048 tokens dependiendo del caso de uso

Algoritmo Completo de Generación

Entrada: prompt, max_length, temperature, top_k, top_p

1. tokens = encode(prompt)
2. for i = 1 to max_length:
    2.1 logits = model(tokens)
    2.2 logits = logits / temperature
    2.3 probs = softmax(logits)
    2.4 probs = apply_top_k(probs, k)
    2.5 probs = apply_top_p(probs, p)
    2.6 next_token = sample(probs)
    2.7 if next_token == EOS: break
    2.8 tokens.append(next_token)
3. return decode(tokens)

💡 Recomendaciones de Uso

Para Texto Coherente y Preciso

  • Temperature = 0.7
  • Top-P = 0.9
  • Top-K = 40

Uso: Respuestas de chatbots, documentación, código

Para Texto Creativo

  • Temperature = 1.2
  • Top-P = 0.95
  • Top-K = sin límite

Uso: Cuentos, poesía, brainstorming
