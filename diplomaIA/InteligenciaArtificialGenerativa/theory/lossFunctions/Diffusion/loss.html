Funcion de Perdida de Difusion

Prediccion de Ruido

La Idea Central

La perdida mas simple de todas:

El modelo aprende a predecir el ruido que se agrego a una imagen.

Loss = ||epsilon - epsilon_pred||^2

epsilon

El ruido REAL que agregaste

epsilon_pred

El ruido que el modelo PREDICE

El Algoritmo de Entrenamiento

1

Tomar imagen

x_0 del dataset

2

Elegir tiempo t

Aleatorio entre 1 y T

3

Muestrear ruido

epsilon ~ Normal(0, 1)

Crear x_t: x_t = sqrt(alpha_bar_t) * x_0 + sqrt(1 - alpha_bar_t) * epsilon
4

Modelo predice

epsilon_pred = model(x_t, t)

5

Calcular loss

MSE(epsilon, epsilon_pred)

6

Actualizar

Gradiente descendente

Por Que Predecir Ruido?

Opcion A: Predecir imagen

Modelo predice x_0 directamente

  • La imagen tiene distribucion compleja
  • Rango de valores variable
  • Menos estable

Opcion B: Predecir ruido (USADA)

Modelo predice epsilon

  • Ruido siempre es Normal(0,1)
  • Rango predecible
  • Mas estable!

Matematicamente equivalentes, pero predecir ruido es mas estable en la practica.

Por Que Se Pasa t?

El modelo recibe dos entradas: la imagen ruidosa (x_t) y el paso temporal (t).

Analogia: Es como un doctor que necesita saber en que etapa de una enfermedad esta el paciente para recetar el tratamiento correcto.

Comparacion con Otras Perdidas

VAE

Reconstruccion + KL

Dos terminos que balancear

GAN

Loss_D + Loss_G

Dos redes que balancear

Difusion

Solo MSE de ruido

Un termino, muy estable

Codigo en PyTorch

def diffusion_loss(model, x_0, t, scheduler):
    # 1. Muestrear ruido
    epsilon = torch.randn_like(x_0)

    # 2. Crear imagen ruidosa (forma cerrada)
    sqrt_alpha_bar = scheduler.sqrt_alphas_cumprod[t].view(-1, 1, 1, 1)
    sqrt_one_minus = scheduler.sqrt_one_minus_alphas_cumprod[t].view(-1, 1, 1, 1)

    x_t = sqrt_alpha_bar * x_0 + sqrt_one_minus * epsilon

    # 3. Modelo predice el ruido
    epsilon_pred = model(x_t, t)

    # 4. Perdida: MSE entre ruido real y predicho
    loss = F.mse_loss(epsilon_pred, epsilon)

    return loss

Resumen para el Parcial

Perdida: L = ||epsilon - epsilon_pred||^2 (MSE de ruido)

Forma cerrada: x_t = sqrt(alpha_bar_t) * x_0 + sqrt(1-alpha_bar_t) * epsilon

Input del modelo: (x_t, t) - imagen ruidosa + paso temporal

Por que ruido: Siempre es Normal(0,1), mas estable que predecir imagen

Ventaja: Un solo termino, muy estable, sin balanceo
