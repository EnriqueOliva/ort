Funcion de Perdida del VAE

Evidence Lower Bound (ELBO)

La Idea Central

La perdida del VAE tiene dos partes que trabajan juntas:

Loss = Reconstruction Loss + KL Divergence
Reconstruccion

Que la copia se parezca al original

+
KL Divergence

Que Z sea Normal(0,1)

Analogia: Imagina que entrenas a alguien para hacer resumenes de libros:

  • Reconstruccion: "Tu resumen debe permitir recrear el libro original"
  • KL: "Tus resumenes deben seguir un formato estandar"

Los Dos Componentes

Reconstruction Loss

Objetivo: Que la imagen reconstruida sea igual a la original.

Opcion A - MSE:

L_recon = ||x - x_hat||^2

Opcion B - BCE:

L_recon = -Sum[ x_i * log(x_hat_i) ]

Cuando usar cada una:

  • MSE: Imagenes con valores continuos
  • BCE: Imagenes binarias (0 o 1)

Si solo usaramos esto: El modelo podria "memorizar" los datos sin aprender una representacion util.

KL Divergence

Objetivo: Que la distribucion del espacio latente sea similar a N(0, 1).

D_KL = 1/2 * Sum(mu^2 + sigma^2 - log(sigma^2) - 1)

Que empuja:

  • mu hacia 0: Las medias se centran
  • sigma hacia 1: Las varianzas se normalizan

Si solo usaramos esto: El modelo ignoraria los datos y produciria ruido aleatorio.

Por Que Necesitamos AMBAS

Codigo en PyTorch

def vae_loss(x, x_recon, mu, log_var):
    # Reconstruction loss (BCE o MSE)
    recon_loss = F.binary_cross_entropy(x_recon, x, reduction='sum')

    # KL Divergence: -0.5 * sum(1 + log(sigma^2) - mu^2 - sigma^2)
    kl_loss = -0.5 * torch.sum(1 + log_var - mu.pow(2) - log_var.exp())

    # Total loss
    return recon_loss + kl_loss

Por que log_var en lugar de sigma?

  • sigma debe ser positivo, pero log(sigma^2) puede ser cualquier numero real
  • Es mas estable numericamente para entrenar
  • Para obtener sigma: sigma = exp(log_var / 2)

Beta-VAE: Controlando el Balance

Una variante popular usa un factor beta para controlar el peso de la KL:

Loss = Reconstruction + beta * KL

beta < 1

Mejor reconstruccion

Menos organizacion

beta = 1

VAE estandar

Balance normal

beta > 1

Mejor organizacion

Disentanglement

Resumen para el Parcial

ELBO: Reconstruction Loss + KL Divergence

Reconstruction: ||x - x_hat||^2 o BCE - mide que tan bien se reconstruye

KL Divergence: 1/2 * Sum(mu^2 + sigma^2 - log(sigma^2) - 1) - empuja mu->0 y sigma->1

Beta-VAE: Loss = Recon + beta * KL - controla el balance
