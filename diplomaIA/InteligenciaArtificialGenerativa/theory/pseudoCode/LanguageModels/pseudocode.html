PREGUNTA DEL PARCIAL

"Implementa en pseudocodigo el proceso de generacion de secuencias de un LM"

secuencia = prompt_inicial REPETIR: logits = LM(secuencia) probs = softmax(logits) token = muestrear(probs) SI token == END: SALIR secuencia = secuencia + token RETORNAR secuencia

1. GENERACION AUTORREGRESIVA (Completo)

FUNCION generar_secuencia(prompt, max_tokens): secuencia = tokenizar(prompt) MIENTRAS len(secuencia) < max_tokens: # 1. Pasar secuencia al modelo logits = LM(secuencia) # 2. Convertir logits a probabilidades probabilidades = softmax(logits) # 3. Muestrear siguiente token siguiente_token = muestrear(probabilidades) # 4. Verificar fin de secuencia SI siguiente_token == END_OF_SEQUENCE: BREAK # 5. Concatenar y repetir secuencia = secuencia + [siguiente_token] RETORNAR detokenizar(secuencia)

3. PUNTO CLAVE: Deterministico vs Estocastico

El LM es DETERMINISTICO
Misma entrada → Misma distribucion de probabilidad (siempre)
La GENERACION es ESTOCASTICA
Porque muestreamos de esa distribucion → puede dar resultados diferentes

4. EJEMPLO PASO A PASO

Paso 1

LM("Hola, como") → {"estas": 0.35, "andas": 0.15, ...}
Muestrear → "estas"
Secuencia: "Hola, como estas"

Paso 2

LM("Hola, como estas") → {"?": 0.40, "hoy": 0.20, ...}
Muestrear → "?"
Secuencia: "Hola, como estas?"

Paso 3

LM("Hola, como estas?") → {END: 0.60, "Yo": 0.15, ...}
Muestrear → END
FIN

RESUMEN ULTRA-CORTO

MIENTRAS no termine: probs = softmax(LM(secuencia)) token = muestrear(probs) SI token == END: terminar secuencia = secuencia + token
