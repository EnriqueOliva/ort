Pseudocodigo GAN

1. ENTRENAMIENTO (Alternado)

PASO 1: ENTRENAR DISCRIMINADOR (D)
# Imagenes REALES -> etiqueta 1 pred_real = D(imagenes_reales) loss_D_real = BCE(pred_real, etiqueta=1) # Imagenes FALSAS -> etiqueta 0 z = muestrear_normal(0, 1) imagenes_falsas = G(z) pred_fake = D(imagenes_falsas.detach()) # IMPORTANTE! loss_D_fake = BCE(pred_fake, etiqueta=0) # Actualizar solo D loss_D = loss_D_real + loss_D_fake backprop(loss_D) actualizar_pesos_D()
.detach() corta el gradiente. Asi NO actualizamos G cuando entrenamos D.
PASO 2: ENTRENAR GENERADOR (G)
z = muestrear_normal(0, 1) imagenes_falsas = G(z) pred_fake = D(imagenes_falsas) # TRUCO DEL FLIPEO: etiqueta=1 aunque son falsas loss_G = BCE(pred_fake, etiqueta=1) backprop(loss_G) actualizar_pesos_G()
Truco del Flipeo: Decimos que las falsas son "reales" (etiqueta=1) para que G aprenda a enganar a D.

2. GENERACION

# Muestrear ruido del espacio latente z = muestrear_normal(0, 1) # Generar imagen con el Generador imagen_nueva = G(z) RETORNAR imagen_nueva

3. PUNTOS CLAVE

Balance D vs G

Si D es muy bueno: G no aprende
Si G es muy bueno: D no aprende

Solucion: lr_D < lr_G

Si pierde balance:

Es mejor reiniciar el entrenamiento desde cero que intentar recuperarse.
