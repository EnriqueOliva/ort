1. ENTRENAMIENTO

PARA cada imagen x_0 del dataset: # 1. Elegir tiempo aleatorio t = random(1, T) # 2. Agregar ruido (forma cerrada) epsilon = muestrear_normal(0, 1) x_t = sqrt(alpha_bar[t]) * x_0 + sqrt(1 - alpha_bar[t]) * epsilon # 3. Predecir el ruido epsilon_pred = modelo(x_t, t) # 4. Perdida: MSE entre ruido real y predicho loss = MSE(epsilon, epsilon_pred) # 5. Actualizar pesos backprop(loss)
Forma Cerrada: Puedes ir directo de x₀ a cualquier x_t sin pasar por pasos intermedios.

2. GENERACION (Reverse Process)

FUNCION generar_imagen(): # Partir de ruido puro x = muestrear_normal(0, 1) # Mismo tamano que imagen # Quitar ruido paso a paso PARA t = T hasta 1: # Predecir el ruido en este paso epsilon_pred = modelo(x, t) # Quitar un poco de ruido x = quitar_ruido(x, epsilon_pred, t) RETORNAR x # Imagen limpia

T = 1000 pasos tipicamente

La generacion es lenta porque hay que recorrer todos los pasos

3. PUNTOS CLAVE

El modelo necesita t: Para saber cuanto ruido hay en la imagen actual
Predecir ruido (no imagen): Mas estable porque el ruido siempre es Normal(0,1)
Espacio latente = mismo tamano: A diferencia de VAE/GAN, el ruido tiene el mismo tamano que la imagen

RESUMEN PARA EL PARCIAL

# ENTRENAR: t = random epsilon = ruido ~ Normal(0,1) x_t = sqrt(ᾱ_t) * x_0 + sqrt(1-ᾱ_t) * epsilon loss = MSE(epsilon, modelo(x_t, t)) # GENERAR: x = ruido_puro PARA t = T hasta 1: x = quitar_ruido(x, modelo(x, t), t) RETORNAR x
