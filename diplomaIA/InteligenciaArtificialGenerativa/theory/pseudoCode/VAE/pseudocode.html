Pseudocodigo VAE

1. ENTRENAMIENTO

PARA cada imagen X del dataset: # ENCODER: comprimir a espacio latente mu, sigma = Encoder(X) # REPARAMETRIZATION TRICK epsilon = muestrear_normal(0, 1) Z = mu + sigma * epsilon # DECODER: reconstruir X_reconstruida = Decoder(Z) # CALCULAR PERDIDA Loss_recon = MSE(X, X_reconstruida) Loss_KL = -0.5 * suma(1 + log(sigma2) - mu2 - sigma2) Loss_total = Loss_recon + Loss_KL # ACTUALIZAR PESOS backprop(Loss_total) actualizar_pesos()

2. GENERACION

# Muestrear del espacio latente (ruido) Z = muestrear_normal(0, 1) # Decodificar para crear imagen nueva X_nueva = Decoder(Z) RETORNAR X_nueva

Generacion = Solo el Decoder

No necesitas el Encoder para generar. Solo muestreas ruido y lo decodificas.

3. REPARAMETRIZATION TRICK

Problema: No podemos hacer backprop a traves de un muestreo aleatorio.

MAL (no diferenciable)

Z = muestrear(mu, sigma) # El gradiente no puede pasar

BIEN (diferenciable)

epsilon = muestrear(0, 1) Z = mu + sigma * epsilon # Gradiente pasa por mu y sigma

RESUMEN PARA EL PARCIAL

# ENTRENAR: Z = Encoder(X) -> mu, sigma Z = mu + sigma * ruido X' = Decoder(Z) Loss = Reconstruccion + KL # GENERAR: Z = ruido ~ Normal(0,1) X_nueva = Decoder(Z)
